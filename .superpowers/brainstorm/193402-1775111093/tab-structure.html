Bot Server Management — Tab Navigation

The management page now has 3 tabs. Does this structure work for you?

1

🤖 Bots (existing)

Split panel: bot list on left, selected bot details and quick actions on right. View and control all bots.

2

⚙️ Server Config

Display server parameters (port, host, MC server, LLM URL, retry settings, etc.). Show current values with edit capability where supported.

3

📋 Logs

Live log viewer reading from bot_server.log. Auto-refresh every 5 seconds. Filter by level (log/error/warn). Scrollable with timestamps.
