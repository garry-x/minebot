Bot Management Page — Layout Options

How should the bot management page look? Choose the layout that best fits your workflow.

A

Table View

A data table listing all bots with columns: Status, Username, Mode, Health, Position, Actions. Click a row to select, buttons to stop/start/remove each bot.

B

Card Grid View

Each bot shown as a card with status dot, username, health/food bars, position, and action buttons. Responsive grid (3-4 cards per row).

C

Split Panel

Left: bot list (compact). Right: selected bot's detailed controls and monitoring. Like a master-detail view.
