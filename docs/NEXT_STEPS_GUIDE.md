## 🛠️ **新开发者入门指南**

### 环境设置
```bash
# 1. 克隆项目
git clone <repository>
cd minebot

# 2. 安装依赖
npm install

# 3. 配置环境
cp .env.example .env
# 编辑.env文件

# 4. 初始化开发环境
npm run build
```

### 开发命令
```bash
# 启动开发服务器
npm run dev

# 运行测试
npm test

# 运行测试带覆盖率
npm test -- --coverage

# 代码检查
npm run lint  # 需要添加package.json脚本

# 代码格式化
npx prettier --write "**/*.js"
```

### 提交代码
```bash
# 1. 确保测试通过
npm test

# 2. 格式化代码
npx prettier --write "**/*.js"

# 3. 提交（遵循规范）
git add .
git commit -m "feat(bot): 添加新行为功能"
```
